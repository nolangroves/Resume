# Nolan Groves — Resume

View online: **https://nolangroves.github.io/Resume/**

Built automatically from `Resume.tex` on every push to main. The PDF is committed to the repo at `Resume.pdf`.

## Variants

Per-company variants live in `variants/`. Each file defines only what differs (typically just the summary), then calls `\input{Resume}`. CI builds any `.tex` file found there.
